# Tic Tac Toe Game

A simple implementation of the classic Tic Tac Toe game using HTML, CSS, and JavaScript.

## Features

- User v/s Computer.
- Clear indication of the game's status.
- Reset button to start a new game.

## Technologies Used

- HTML
- CSS
- JavaScript
